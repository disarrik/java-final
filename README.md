# VK API Client

Клиентское приложение для работы с VK API, которое анализирует статистику пользователей в указанном городе.

## Сборка Docker образа

Для сборки Docker образа выполните следующую команду в корневой директории проекта:

```bash
docker build -t vk-api-client .
```

Эта команда создаст Docker образ с именем `vk-api-client`.

## Запуск контейнера

### Запуск с переменной окружения

```bash
docker run -e vk.api.token=ВАШ_ТОКЕН vk-api-client
```

## Логирование

Приложение записывает логи в файл `logs/vk-api-client.log` внутри контейнера. Для сохранения логов на хост-машине можно использовать volume:

```bash
docker run -e vk.api.token=ВАШ_ТОКЕН -v $(pwd)/logs:/app/logs vk-api-client
```

## Функциональность

Приложение выполняет следующие действия:

1. Ищет пользователей в указанном городе
2. Находит пользователя с наибольшим количеством друзей среди пользователей с открытым профилем
3. Собирает статистику по друзьям, группам и постам найденного пользователя
4. Выводит отчет в консоль
